This package is mainly for HttpServer.

1. The HttpServer.java is used to connect to remote server, it use the WorkThread.java to do it.
2. The WorkThread.java is the only HTTP out port for the Application.
3. The SendOnline.java is used to send a heart-beat signal to the remote server.
4. The GetIP.java is used to find the IP in the phone. It also can use in the PC.
